## Description

An introduction to programming through creative practice. Students will learn fundamental programming concepts—variables, loops, conditionals, functions—while creating visual and interactive artwork using p5.js.

## Learning Objectives

By the end of this course, students will be able to:

1. Write programs that generate visual output
2. Create interactive experiences responding to user input
3. Work with images, video, and sound programmatically
4. Develop a portfolio of creative coding projects
