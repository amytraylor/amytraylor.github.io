Image & Download

Loading and processing images with p5.js, then saving the result.

Try this:

  • Select different filters from the dropdown
  • Click "Save PNG" or "Save JPG" to download
  • Move mouse to adjust filter intensity

This sketch demonstrates:

  • loadImage() in preload()
  • Built-in filters: filter(INVERT), etc.
  • saveCanvas() for downloading
  • Using local assets (see assets/ folder)
